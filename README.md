# InstagramDL
> Download Instagram images, and stories simply


### Download only a post
```bash
python3 . --post CPkgc--NkhZ
```

It downloads contents from [here](https://www.instagram.com/p/CPkgc--NkhZ/)

### Download all posts from a user
```bash
python3 . --user yoo__sha
```

It downloads all contents from [here](https://instagram.com/yoo__sha)

### Download story from a user
```bash
python3 . --story yoo__sha
```

and then, a Chrome tab will open. Login with your IG account. (It doesn't take your credentials)
